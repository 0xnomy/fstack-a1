# Portfolio Card Assignment

This assignment focuses on building a responsive, beautifully styled "Project Showcase Card" using strictly pure HTML and CSS, with zero JavaScript or frameworks. Over the course of the project, we focused on practicing semantic HTML structure, advanced CSS properties (like flexbox, media queries, and transform transitions), and even built a fully interactive requirements checklist using native `<details>` and `<summary>` tags.

A massive thank you to **Laraib Noor** for teaching us all about full-stack development! Your instruction laid the perfect foundation for learning how to methodically build and structure modern web interfaces from scratch.
